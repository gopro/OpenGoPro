    Open GoPro

    An open source interface specification to communicate with a GoPro camera with accompanying demos and tutorials.

    View on Github

    The Open GoPro API is the primary way for users to interact with a GoPro camera. The camera provides interfaces to WiFi and Bluetooth Low Energy that allow users to perform command, control, and query actions including, but not limited to:

    • Get camera state
    • Load presets
    • Change settings
    • Digital Zoom
    • Start and stop encoding
    • Get media list
